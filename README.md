# Shutdown computer

A package shutdown computer

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install shutdown_computer.

```bash
pip install shutdown_computer
```

## Usage

```python
from shutdown_computer import shutdown

shutdown()
```
